Quiz Instructions

How this quiz works

This quiz demonstrates multiple JavaScript concepts. The questions are stored separately in a JSON file, which means they are not hard-coded into the HTML page. JavaScript loads them dynamically, randomises the order, renders the options, and then checks the user’s answers when the form is submitted.

After submitting, the page calculates the score, percentage, and pass/fail result. It also explains each question so the quiz becomes a learning tool, not just a scoring tool. Each attempt is saved in local storage so progress can be tracked over time.
